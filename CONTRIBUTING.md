# Contributing to OracleNet

## Ways to contribute

### 1. Use it and report issues
Connect to our MCP servers and report any issues you encounter.

### 2. Build on top of it
- Build agents that use OracleNet tools
- Create workflows using the Conductor
- Integrate with the escrow contract

### 3. Improve documentation
- Fix typos, clarify explanations
- Add examples in other languages
- Translate docs

### 4. Propose new oracles
Open a Discussion with your idea for a new oracle. What data source? What compliance need?

### 5. Security reports
Found a vulnerability? Email security@feedoracle.io — do NOT open a public issue.

## Code of conduct

Be respectful. Be constructive. We're building infrastructure for regulated industries — accuracy and reliability matter more than speed.
